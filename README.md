## 机制

生物与玩家各有一个 **0-200 精力值(Energy)**。雄性生物精力越高，越主动寻找目标触发CG。
  
游戏内玩家的不同部位 **V、A、M** 分别对应 **雪、肛、口**。

**加速精力增长**：玩家精力 >100 时发射光环（6~20 格），范围内生物精力加速充能。皮肤破损会导致精力充能加速 50 倍。附近有动画进行中 +3 倍。

**CG 匹配条件**（全部满足才触发）：
1. 双方实体类型和性别标签吻合（主动方须 `gender.male`）
2. 环境满足（平地/墙/半砖/水中）
3. 玩家未被保护（如骑马时可能触发保护）
   
## 基础触发方式

| 方式 | 条件 |
|------|------|
| 自然遭遇 | 玩家精力>=100，周围有雄性生物即自动触发 |
| 战败 | 被生物打出致死伤害，生物精力>=70（120s CD，不死） |
| 主动请求 | 玩家精力>=70，手动选目标/动画 |
| PVP 攻击 | 左键攻击玩家或被攻击，按键竞争逃脱 |
