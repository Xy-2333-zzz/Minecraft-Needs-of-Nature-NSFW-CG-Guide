## 触发方式

| 方式 | 条件 |
|------|------|
| 自然遭遇 | 玩家能量>=100，周围有雄性生物即自动触发 |
| 战败 | 被生物打出致死伤害，生物能量>=70（120s CD，不死） |
| 主动请求 | 玩家能量>=70，手动选目标/动画 |
| PVP攻击 | 左键攻击玩家或被攻击，按键竞争逃脱 |
